## 🚀 Project Overview

This project focuses on analyzing PhonePe transaction and insurance data to uncover meaningful insights about digital payment trends, user behavior, and regional performance across India.

The goal is to transform raw JSON data into actionable business insights using data analysis, SQL, and interactive visualization.
## 🎯 Problem Statement

With the increasing adoption of digital payment systems like PhonePe, it is important to understand transaction patterns, user engagement, and regional trends.
## 📂 Project Structure

PhonePe-Insights/
│

├── app.py                # Streamlit dashboard

This project aims to:
- Analyze transaction data across states and quarters
- Identify top-performing regions
- Understand user behavior and transaction growth
- Provide actionable insights for business decisions
- ## ✨ Features

- End-to-End Data Pipeline (ETL)
- Data Cleaning and Wrangling
- Exploratory Data Analysis (15+ Visualizations)
- SQL Database Integration (SQLite)
- Interactive Dashboard using Streamlit
- Business Insights and Recommendations
- ## 🛠 Tech Stack

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- SQL (SQLite)
- Streamlit
- ## 💡 Key Insights

- A few states contribute significantly to total transaction value
- Strong correlation between transaction count and transaction amount
- Digital payment adoption is steadily increasing over time
- Some regions show lower activity, indicating growth opportunities
- High-value transactions suggest potential for premium services
